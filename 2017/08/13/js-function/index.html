JS-Function

2017-08-13
#Js

一、函数的定义
其实函数就是一个Function类型的实例，和其他的引用类型一样有属性和方法。所以函数名实质就是一个对象的引用；

定义函数有3种方法；
1.函数声明

函数声明有一个变量提升的过程，JS引擎会在调用所有代码之前先生成好函数声明的方法。所以即使在函数声明之前调用这个函数，也是可以的

2.函数表达式

这样定义函数，引擎不会对它进行变量声明，而是等到它执行到这个语句的时候他才会定义这个函数，所以在这之前使用会报错,找不到这个函数。

3.构造函数

从这里可以看出，函数就是FUNCTION类型的实例而已，而二次定义函数，实际上是会覆盖掉原来的值

所以函数也没有像Java一样有函数重载之说，如果想模拟函数重载,可以通过typeof和arguments在内部作处理

相当于java中的

二、函数内部属性
在函数内部，有两个内部属性，一个是arguments，是一个类数组对象，用来接收函数的参数；还有一个是this，用来指向该函数的执行的环境对象；

三、函数属性和方法
属性：
1.length 该函数期望接收的参数

2.prototype 该函数作为构造函数时创建的实例所共享的对象，里面包含属性和方法，不可枚举；

所以用第三种方法不无法获取这个属性的

方法：
1.apply(运行的作用域，数组实例或者类数组) //执行函数
2.call(运行的作用域，参数1，参数2，参数3..) //执行函数
3.bind(运行的作用域) //只绑定作用域，不执行
可以通过这三种方法去绑定this的值
